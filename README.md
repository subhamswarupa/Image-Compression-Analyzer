# 🖼️ ImagePress – Image Compression Analyzer

A modern full-stack web application that allows users to compress images in real time, compare the original and compressed versions side-by-side, analyze compression metrics, and download optimized images. Designed with a premium AI-inspired interface, ImagePress makes it easy to find the perfect balance between image quality and file size.

---

## ✨ Features

- 📤 Drag & Drop image upload
- 🖼️ Side-by-Side image comparison
- 🎚️ Live compression quality slider
- 🔄 Real-time image recompression
- 📥 Download compressed images
- 📜 Compression history
- 🌙 Dark & Light mode
- 📱 Fully responsive design

### 📊 Compression Analytics

- Original File Size
- Compressed File Size
- Compression Ratio
- Storage Saved
- Image Dimensions
- Quality Level
- Estimated Quality Loss
- Estimated Loading Speed Improvement
- Compression Efficiency Gauge
- Quality Score
- Storage Comparison Bar

## ⚙️ How It Works

1. Upload an image.
2. Adjust the compression quality using the slider.
3. The image is compressed instantly.
4. Compare the original and compressed versions.
5. Analyze compression metrics.
6. Download the optimized image.

## 💻 Installation


### Install dependencies


```bash
npm run dev
```
